# MoodFlix

Discover movies that match your mood — not just your genre.

A learning project built with Next.js (App Router), TypeScript, and Tailwind CSS, using the TMDB API for movie data.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- TMDB API
